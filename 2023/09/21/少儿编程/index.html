少儿编程推荐

1，http://www.code.org
Code.org是一个非盈利性组织，创建了同名的网站Code.org。该网站旨在支持美国的学生学习计算机科学，随着网站的发展，志愿者也逐渐增多，翻译工作也水到渠成进行着。Code.org期望每一名学生都有机会接触并学习计算机科学，并且提供了许多课程资源，学生可以在线学习编程。课程适合绝大部分的人，只要年龄超过4岁，就可以开始学习计算机科学基础。Code.org还得到了亚马逊、脸书、谷歌、印孚瑟斯基金会、微软等的支持。此外，Code.org网站上有很多编程的在线课程，课程是以一个个小任务来把编程知识融入到游戏中，一个个关卡通关，编程能力就会一步步提升。

2，http://scratch.mit.edu
Scratch是麻省理工学院的“终身幼儿园团队”开发的一种图形化编程工具，是面向青少年的一款模块化、积木化、可视化的编程语言。

模块化和积木化是指，相较于传统编程语言（如Python、Java等），Scratch的代码以一个个模块的形式呈现，每个模块代表一行代码，用户可以通过拖拽的方式将各个模块连接在一起，犹如搭积木一样，大大降低了编程的难度。

可视化则是指，在Scratch中，用户可以直观地看到他们所编写的代码的效果。在右边的舞台区，用户可以对角色或背景进行编程，编程结果非常直观。

总的来说，Scratch是一个非常值得推荐的工具，特别是对于初学者或者青少年来说，它不仅可以帮助他们理解编程的基础概念，还能激发他们对编程的兴趣。考虑到它的易用性、教育价值和创新性，我给Scratch推荐星数为4颗星。

3，http://edu.shaoerbc.org

少儿编程学院可以提供专业的少儿编程教育，致力于激发孩子的科技能力和未来素养，帮助孩子理解和掌握编程语言和技能，提高逻辑思维和创造力。例如，美国麻省理工学院的Scratch，以及英国的Code First: Girls。由于少儿编程学院可以有效地提升孩子对于编程的学习能力和创造能力，我给少儿编程学院的推荐星数为4颗星。

4，https://codecombat.cn/
Code Combat是一款网页编程游戏，玩家在游戏中扮演英雄角色，通过编写代码来控制英雄移动、攻击和防御等行为，与其他玩家进行战斗。

Code Combat采用了一种集成式开发环境，用户可以随时随地使用任何浏览器和设备访问，无需安装任何插件或应用程序，使用基于Scratch和JavaScript的编程语言进行开发，提供了实时多人在线对战、聊天室、合作项目等功能。
Code Combat适合对编程感兴趣的儿童和青少年，可以让他们在游戏中学习编程语言和技能，提高逻辑思维能力和创造力。同时，Code Combat还提供了丰富的成就系统和奖励机制，可以激发孩子的成就感和自信心。

考虑到Code Combat的趣味性、互动性和教育价值，我给Code Combat推荐星数为4颗星。

5，http://www.lightbot.com
Light Bot是一款休闲益智游戏，通过控制机器人前往目的地并点亮灯泡解决关卡问题，非常适合初学者和儿童，能够锻炼玩家的逻辑思维和编程能力。

在游戏中，玩家需要通过编程指令让机器人行动，完成任务并点亮灯泡即可顺利过关。游戏画面简单可爱，音效和音乐也非常有趣。Light Bot拥有超过40个关卡和20个挑战，难度逐渐提高，非常具有挑战性。

此外，Light Bot还提供了多种语言版本，包括中文版，让玩家能够轻松上手并享受不同语言带来的不同乐趣。

综上所述，Light Bot具有休闲益智、简单可爱画面音效、多种语言版本等特点，非常适合初学者和儿童，能够锻炼玩家的逻辑思维和编程能力，是一款非常有趣的游戏。因此，我给Light Bot推荐星数为4颗星。

6，https://www.icode.org.cn/
ICode是一个全球化的非盈利组织，旨在全球范围内推广编程教育。通过区域、国家和国际多形式的赛事，为全球学生创造一个免费、有趣、有效的学习和交流平台。

ICode国际青少年编程竞赛是该组织举办的一项国际比赛，自2019年以来已经举办了多届。该比赛不仅为全球的学生提供了一个展示编程能力的舞台，同时也为他们提供了一个学习编程知识和技能的机会。此外，ICode还与微软、苹果、百度、Unity等大科技公司建立了战略合作伙伴关系，以进一步推动编程教育的发展。

考虑到ICode在推广编程教育方面的积极作用以及其举办的竞赛对全球学生的积极影响，我给ICode推荐星数为4颗星。

7，https://www.rrcoding.cn
是一个在线编程学习平台，旨在让每个人都能够轻松地学习和体验编程。它提供了多种编程语言和课程，包括Scratch、JavaScript、Python等，适合不同年龄段和不同程度的编程学习者。

人人编程的特点在于其亲少儿编程的定位，它能够让孩子在游戏中学习编程，同时能够为孩子提供专业的编程学习路径和指导，帮助孩子逐渐掌握编程技能和逻辑思维。人人编程还提供了丰富的互动和社交功能，如作品展示、交流讨论等，能够激发孩子的学习兴趣和创造力。

考虑到人人编程的亲少儿编程定位、专业的学习路径和丰富的互动功能，我给人人编程推荐星数为4颗星。
